```
서버 아키텍쳐 고민을 깊게 수행하였습니다.

기존에는 소켓 통신을 활용해 진행하려 했으나, 굳이 데이터가 양방향으로 소통할 이유가 없다고 생각했습니다.

이러한 고민들을 통해 서버 아키텍처를 확정할 수 있었고, 서비스 흐름도를 만들 수 있었습니다.
```
