## 🎯 주제
<a name="subject"></a>

**주경 야학**은 몸이 불편하거나 근처에 야학이 없어서 기존 야학에 참여하지 못하는 어르신들을 위한 편리한 교육 플랫폼입니다.

공간적 제약을 해소함으로써 더 많은 어르신에게 교육의 기회를 제공하고, 봉사자의 진입 장벽을 낮추는 것을 목표로 합니다.

**주요 기능:**
- 실시간 수업
- 개별 학습 지원
- 대화를 통한 일기 쓰기 학습
